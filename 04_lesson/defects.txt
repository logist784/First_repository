ID: STR-001
   Заголовок: Метод contains() возвращает некорректный результат для пустого символа

   Описание:
   Метод contains() возвращает True при проверке пустого символа в любой строке,
   включая пустую строку. Согласно ожиданиям тестов, должен возвращать False.

   Шаги:
   1. Вызвать string_utils.contains("test", "")
   2. Вызвать string_utils.contains("", "")
   3. Наблюдать возвращаемое значение

   Ожидаемый результат: False
   Фактический результат: True

   Затронутые тест-кейсы:
   - TestContains.test_contains_negative[test--False]
   - TestContains.test_contains_negative[--False]
